# OSINT Field Note - Employee Credential Exposure Check

Purpose: assess whether corporate email addresses appear in publicly known data
breach compilations so the organization can force resets proactively.

Method: submit corporate domains to a reputable breach-notification service that
operates on already-public datasets. Record which accounts appear and in which
disclosed breach. Do not collect or store the exposed passwords themselves.

Action: for each exposed account, require a password reset and verify multi
factor authentication. Track remediation to closure. This is defensive exposure
management using public information.
